# Coffee-Machine-Simulator
Coffee Machine Simulator â€” A simple Python console app that lets users order espresso, latte, or cappuccino, handles coin-based payments, tracks resources, and calculates profits. Great for practicing control flow, functions, and user input handling.


Features
  Three drink options: Espresso, Latte, and Cappuccino
  
  Ingredient tracking and resource management
  
  Coin-based payment system with change calculation
  
  Profit tracking and machine status reporting
  
  Graceful handling of insufficient resources or payment


How It Works
  User selects a drink or enters report or off
  
  Machine checks if enough ingredients are available
  
  User inserts coins (quarters, dimes, nickels, pennies)
  
  If payment is sufficient, the drink is made and resources are updated
  
  Machine prints a report showing remaining resources and profit

Requirements
  Python 3.6 or higher
  No external libraries required
